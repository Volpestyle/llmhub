## Model inputs

- voice_id (optional): Voice ID for speech synthesis (if using text and not audio) (string)
- text (optional): Text content for lip sync (if not using audio) (string)
- video_id (optional): ID of a video generated by Kling. Cannot be used with video_url. (string)
- video_url (optional): URL of a video for lip syncing. It can be an .mp4 or .mov file, should be less than 100MB, with a duration of 2-10 seconds, and a resolution of 720p-1080p (720-1920px dimensions). Cannot be used with video_id. (string)
- audio_file (optional): Audio file for lip sync. Must be .mp3, .wav, .m4a, or .aac and less than 5MB. (string)
- voice_speed (optional): Speech rate (only used if using text and not audio) (number)
